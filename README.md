# DESTINY: The Simulation

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)
![GitHub Deploy](https://img.shields.io/badge/Deployment-GitHub%20Pages-brightgreen)

**DESTINY: The Simulation** is an interactive, choice-driven life narrative game. From your first breath to your final legacy, every decision you make shapes your unique destiny.

---

## 🎮 How to Play
1. **Start:** You begin as an infant.
2. **Choose:** Click buttons to influence your character's path.
3. **Grow:** Navigate through key life stages (Childhood, Adolescence, Mid-Life).
4. **Destiny:** Your stats (Boldness & Patience) and decisions lead to one of many unique ending paths.

## 🚀 Features
*   **Dynamic Storytelling:** Multiple branching paths based on player choice.
*   **Stat Tracking:** Your personality (Boldness/Patience) evolves with you.
*   **Modern UI:** Glass-morphism design with responsive Tailwind integration.
*   **Instant Restart:** Try again and pursue a different destiny.
*   **Automatic Deployment:** CI/CD pipeline set up via GitHub Actions.

## 🛠️ Built With
*   **Core:** Vanilla JavaScript (ES Modules)
*   **UI/Styling:** Tailwind CSS & Custom CSS3 Animations
*   **Deployment:** GitHub Pages / GitHub Actions

## 📦 Getting Started
1. **Clone the repo:** `git clone https://github.com/TheStrongestOfTomorrow/Text-game.git`
2. **Run locally:** Open `index.html` in a live server environment (recommended for ES Module support).
3. **Push to deploy:** Simply push your changes to the `main` branch to trigger the auto-deployment workflow.
